## GCP에 배포

<Note>
  `weave deploy`는 기기에 `gcloud`가 설치되고 구성되어 있어야 합니다. `weave deploy gcp`는 명령줄 인수로 직접 지정되지 않은 경우 미리 구성된 설정을 사용합니다.
</Note>

Weave 모델에 대한 Weave 참조가 있으면 다음을 실행할 수 있습니다:

```
weave deploy gcp <ref>
```

모델을 제공하는 gcp 클라우드 함수를 배포하기 위해. 배포의 마지막 줄은 다음과 같이 보일 것입니다 `Service URL: <PATH_TO_MODEL>`. 방문하세요 `<PATH_TO_MODEL>/docs` 모델과 상호작용하기 위해!

실행

```
weave deploy gcp --help
```

모든 명령줄 옵션을 보려면.
