---
- title: 서빙
- description: Weave 작업과 모델을 API 엔드포인트로 노출
---

Weave 모델에 대한 Weave 참조가 있으면 다음을 실행할 수 있습니다:

```
weave serve <ref>
```

해당 모델에 대한 FastAPI 서버를 실행합니다. [http://0.0.0.0:9996/docs](http://0.0.0.0:9996/docs)를 방문하여 모델을 대화식으로 쿼리할 수 있습니다.

## FastAPI 설치

```bash
pip install fastapi uvicorn
```

## 모델 서빙

터미널에서 다음을 호출하세요:

```bash
weave serve <your model ref>
```

모델 참조를 얻으려면 모델로 이동하여 UI에서 복사하세요. 다음과 같은 형식이어야 합니다:

`weave:///your_entity/project-name/YourModel:<hash>`

사용하려면 Swagger UI 링크로 이동하여 predict 엔드포인트를 클릭한 다음 "Try it out!"을 클릭하세요.
