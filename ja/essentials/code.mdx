## 基本

### インラインコード

コードとして `word` または `phrase` を示すには、バッククォート(\`)で囲みます。

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

### コードブロック

使用 [フェンスドコードブロック](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) は、3つのバッククォートでコードを囲み、先頭のティックの後にスニペットのプログラミング言語を記述することで、構文ハイライトを得ることができます。オプションで、プログラミング言語の後にコードの名前を書くこともできます。

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
