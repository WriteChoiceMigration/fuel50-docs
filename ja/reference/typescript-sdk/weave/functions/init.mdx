# Function: init()

> **init**(`project`, `settings`?): `Promise`\<[`WeaveClient`](../classes/WeaveClient.mdx)>

Weaveクライアントを初期化します。これはweaveトレースが機能するために必要です。

## パラメータ

• **project**: `string`

W\&Bプロジェクト名（projectまたはentity/projectの形式）。

• **settings?**: `Settings`

(オプション) Weaveトレース設定

## 戻り値

`Promise`\<[`WeaveClient`](../classes/WeaveClient.mdx)>

初期化されたWeaveクライアントに解決されるプロミス。

## 例外

初期化が失敗した場合
