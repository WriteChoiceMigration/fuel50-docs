<Info>
  **前提条件**：続行する前に Node.js（バージョン19以上）をインストールしてください。<br />続行する前に `docs.json` にアップグレードし、レガシーの `mint.json` ファイルを削除してください。
</Info>

お使いのオペレーティングシステムに Mintlify をインストールして実行するには、次の手順に従ってください：

**ステップ 1**：Mintlify をインストールします：

<CodeGroup>
  ```bash npm
  npm i -g mintlify
  ```

  ```bash yarn
  yarn global add mintlify
  ```
</CodeGroup>

**ステップ 2**：docs ディレクトリ（`docs.json` ファイルがある場所）に移動し、次のコマンドを実行します：

```bash
mintlify dev
```

ドキュメントのローカルプレビューが `http://localhost:3000` で利用可能になります。

### カスタムポート

デフォルトでは、Mintlify はポート 3000 を使用します。Mintlify が実行されるポートは `--port` フラグを使用してカスタマイズできます。例えば、Mintlify をポート 3333 で実行するには、次のコマンドを使用します：

```bash
mintlify dev --port 3333
```

すでに使用中のポートで Mintlify を実行しようとすると、次に利用可能なポートが使用されます：

```md
Port 3000 is already in use. Trying 3001 instead.
```

## Mintlify のバージョン

各 CLI リリースは Mintlify の特定のバージョンに関連付けられていることに注意してください。ローカルウェブサイトが本番バージョンと一致しない場合は、CLI を更新してください：

<CodeGroup>
  ```bash npm
  npm i -g mintlify@latest
  ```

  ```bash yarn
  yarn global upgrade mintlify
  ```
</CodeGroup>

## リンクの検証

CLI はドキュメント内で作成された参照リンクの検証に役立ちます。壊れたリンクを特定するには、次のコマンドを使用します：

```bash
mintlify broken-links
```

## コードフォーマット

IDE で MDX を認識してフォーマットするための拡張機能の使用をお勧めします。VSCode ユーザーの場合は、構文ハイライト用の [MDX VSCode extension](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx) と、コードフォーマット用の [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) を検討してください。

## トラブルシューティング

<AccordionGroup>
  <Accordion title="Error: Could not load the &#x22;sharp&#x22; module using the darwin-arm64 runtime">
    これは node の古いバージョンが原因かもしれません。次の操作を試してください：

    1. 現在インストールされている mintlify のバージョンを削除します： `npm remove -g mintlify`
    2. Node v19 以上にアップグレードします。
    3. mintlify を再インストールします： `npm install -g mintlify`
  </Accordion>

  <Accordion title="Issue: Encountering an unknown error">
    Solution: Go to the root of your device and delete the \~/.mintlify folder. Afterwards, run `mintlify dev` を再度実行します。
  </Accordion>
</AccordionGroup>

CLI バージョンで何が変更されたか知りたいですか？ [CLI の変更履歴をチェックしてください。](https://www.npmjs.com/package/mintlify?activeTab=versions)
