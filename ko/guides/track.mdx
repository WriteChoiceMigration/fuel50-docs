## 작동 방식

몇 줄의 코드로 머신 러닝 실험을 추적하세요:

1. W\&B [W\&B run](/guides/runs)을 생성하세요.
2. 학습률이나 모델 유형과 같은 하이퍼파라미터 딕셔너리를 구성([`wandb.config`](/guides/track/config))에 저장하세요.
3. 훈련 루프에서 정확도와 손실과 같은 메트릭을 로깅([`wandb.log()`](/guides/track/log))하세요.
4. 모델 가중치나 예측 테이블과 같은 실행 결과를 저장하세요.

다음 코드는 일반적인 W\&B 실험 추적 워크플로우를 보여줍니다:

```python
# Start a run.

# When this block exits, it waits for logged data to finish uploading.

with wandb.init(entity="", project="my-project-name") as run:

  # Run your experiment code.
  for epoch in range(num_epochs):

    run.log({"loss": loss})
```

## 시작하기

사용 사례에 따라 W\&B Experiments를 시작하기 위해 다음 리소스를 살펴보세요:

* 데이터셋 아티팩트를 생성, 추적 및 사용하는 데 사용할 수 있는 W\&B Python SDK 명령의 단계별 개요를 보려면 [W\&B Quickstart](/quickstart)를 읽어보세요.

* 다음 방법을 배우기 위해 이 장을 살펴보세요:

  * 실험 생성하기
  * 실험 구성하기
  * 실험에서 데이터 로깅하기
  * 실험 결과 보기

* W\&B API 참조 가이드 내의 [W\&B Python Library](/ref/python/)를 [W\&B API Reference Guide](/ref/)에서 살펴보세요.
